Om du vill använda ns API, så behöver du bara ladda ner (dvs skicka ett HTTP GET request till) datakatalogen när du beöver information om vilka paket hanterar. Denna fil innehåller information om alla paket i systemet. Om du behöver än mer information om ett paket än vad som kan hittas i datakatalogsfilen, sök rätt på det paketet, och följ länken som erbjuds i "<details>"-tagen för det paketet. Denna länk leder dig till en fil som innehåller ytterligare information om ditt paket.
