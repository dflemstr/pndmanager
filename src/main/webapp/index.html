Welcome to the online PND package manager!
Here, you can browse and download PND packages published by other users for your OpenPandora system!

The manager only allows the editing of packages for logged-in users. Please log in to make your session interactive.

We like open technologies, and therefore use the OpenID user authorization system. If you want to log in on this site, please get an OpenID account from a provider. You most likely already have an OpenID account if you use some of the big social media sites (get more information here); otherwise, you can get an OpenID from for example claimID
(note: do some research, claimID might not be the best solution for you).

To browse and contribute to the PND repository, look in the menu to the left.
